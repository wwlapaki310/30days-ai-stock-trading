# 生成AIに従って30日間株取引してみた

## 📖 プロジェクト概要

本プロジェクトは、生成AIのアドバイスに従って30日間株取引を行い、その過程と結果を記録・分析し、最終的に書籍化することを目指しています。

## 🎯 目的

- 生成AIを活用した株取引の可能性と限界を探る
- 日々の取引記録を詳細に残す
- AIとの対話を通じた投資判断プロセスを可視化する
- 実践的な知見を書籍としてまとめる

## ⚠️ 重要な制約事項

### 日本株は100株単位での取引

日本株式市場では、**100株を1単元**として取引されます。これは本プロジェクトの戦略に大きな影響を与えます：

- ❌ **不可能**: 50株だけ売却、150株購入など
- ✅ **可能**: 100株、200株、300株...の単位での売買のみ

**戦略への影響**:
- 「半分利確」「一部売却」などの柔軟な対応が困難
- 全株ホールド or 全株売却の二択になるケースが多い
- 利確判断がより難しくなる
- ポジションサイズの重要性が増す

**対策**:
- 1銘柄につき最低200株保有することで「段階的売却」が可能に
- 100株×2で保有すれば、利確・損切りの柔軟性が向上
- リスク管理の観点からも複数単元保有を推奨
