# 生成AIに従って30日間株取引してみた

## 📖 プロジェクト概要

本プロジェクトは、生成AIのアドバイスに従って30日間株取引を行い、その過程と結果を記録・分析し、最終的に書籍化することを目指しています。

## 🎯 目的

- 生成AIを活用した株取引の可能性と限界を探る
- 日々の取引記録を詳細に残す
- AIとの対話を通じた投資判断プロセスを可視化する
- 実践的な知見を書籍としてまとめる

## 📝 取引ルール

### 100株単位取引
日本株は**100株を1単元**として取引します。本プロジェクトでは以下のルールで運用：

- **最小取引単位**: 100株
- **買い増し・売却**: すべて100株単位
- 例：「50株売却」「150株購入」は不可能
- 例：「100株売却」「200株購入」は可能

このため、AIが「150株」などの提案をした場合は、100株単位に調整して実行します。
