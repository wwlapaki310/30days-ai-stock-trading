# 生成AIに従って30日間株取引してみた

## 📖 プロジェクト概要

本プロジェクトは、生成AIのアドバイスに従って30日間株取引を行い、その過程と結果を記録・分析し、最終的に書籍化することを目指しています。

## 🎯 目的

- 生成AIを活用した株取引の可能性と限界を探る
- 日々の取引記録を詳細に残す
- AIとの対話を通じた投資判断プロセスを可視化する
- 実践的な知見を書籍としてまとめる

## 📁 プロジェクト構造

```
.
├── README.md           # このファイル
├── daily-logs/         # 日々の取引記録
├── notes/              # 気づきやメモ
├── manuscript/         # 原稿ドラフト
├── data/               # データや分析結果
└── prompts/            # 使用したプロンプト集
```

## 📅 実施期間

- 開始日: 2025年10月2日
- 予定期間: 30日間
- 記録方法: 毎日の取引内容、AI との対話内容、市場分析、感想などを記録

## 🤖 使用するAI

本プロジェクトでは、複数の生成AIを使用して銘柄選定を行い、その結果を比較検証します。

- **Claude Sonnet 4.5** (Anthropic)
- **Gemini Pro** (Google)
- **ChatGPT** (OpenAI)

## 💡 10日間で学んだこと

### 1. AIの特性理解と活用法
- **Claude**: 市場トレンド分析、マクロ視点、リスク管理重視
- **Gemini**: テーマ集中、保守的判断、損切り推奨
- **ChatGPT**: バランス重視、戦略的思考、入れ替え提案

### 2. AI多数決の有効性
**威力**:
- 感情を排除した客観的判断
- 複数の視点で多角的に判断
- 意見が分かれた場合の合理的決断

**Day 10の実践**:
- IHI売却: 全AI一致（3対0）
- 野村マイクロ保有: 2対1（ChatGPT/Gemini保有 vs Claude売却）
- アドバンテスト購入: 全AI一致（3対0）

### 3. 損切りと利確のバランス
**損切りの実践**:
- Day 7: 石川製作所-5.49%で機械的損切り
- Day 10: IHI-6.58%で規律ある損切り
- 「損小利大」の原則を徹底

**利確の判断**:
- Day 9: サムコ+7.09%で確実な利益確保
- Day 10: 野村マイクロ+10.75%でもトレンド優先で保有継続

### 4. テーマ性の重要性と限界
**成功したテーマ**:
- Physical AI: グローバルで長期的な成長テーマ
- 半導体: AI需要という本質的な需要に支えられる

**失敗したテーマ**:
- 防衛: 政治テーマは短期的で不安定
- サイバーセキュリティ: 一過性のブーム

### 5. ポートフォリオ構築の進化
**Day 1-3**: セクター分散型（バラバラ）
**Day 4-6**: 3軸分散型（半導体・防衛・Physical AI）
**Day 7-9**: 攻めと守りのバランス型
**Day 10**: 2大テーマ集中型（半導体60%、Physical AI 40%）

### 6. 「ルール vs トレンド」の高度な判断
**Day 10の野村マイクロ判断**:
- ルール: +10%で利確（Claude）
- トレンド: セクター全体の勢いに乗る（ChatGPT、Gemini）
- 結論: トレンドを優先し、より大きな利益を狙う
