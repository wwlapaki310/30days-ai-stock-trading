# 生成AIに従って30日間株取引してみた

## 📖 プロジェクト概要

本プロジェクトは、生成AIのアドバイスに従って30日間株取引を行い、その過程と結果を記録・分析し、最終的に書籍化することを目指しています。

## 🎯 目的

- 生成AIを活用した株取引の可能性と限界を探る
- 日々の取引記録を詳細に残す
- AIとの対話を通じた投資判断プロセスを可視化する
- 実践的な知見を書籍としてまとめる

## 📁 プロジェクト構造

```
.
├── README.md           # このファイル
├── daily-logs/         # 日々の取引記録
├── notes/              # 気づきやメモ
├── manuscript/         # 原稿ドラフト
├── data/               # データや分析結果
└── prompts/            # 使用したプロンプト集
```

## 📅 実施期間

- 開始日: 2025年10月2日
- 予定期間: 30日間
- 記録方法: 毎日の取引内容、AI との対話内容、市場分析、感想などを記録

### 2. AI多数決の実践（Day 11）
**Geminiの主張**:
- アドバンテスト売却（-1.45%損切り）
- IHIに大量買い増し
- 短期決戦で高市総理就任の恩恵を最大化

**ChatGPT・Claudeの主張**:
- 三菱重工を新規購入
- 全保有銘柄継続
- 防衛2本柱でリスク分散

**最終決定**: ChatGPT・Claude案を採用（2-1）

**理由**:
- IHI単独集中は材料出尽くしリスク大
- アドバンテストは損切りラインではない
- 三菱重工でリスク分散しつつ防衛テーマ強化
- 1ヶ月チャレンジの中盤戦、まだ時間がある

### 3. 「Physical AIはグローバルトレンド」の発見
**重要な気づき**:
- 安川電機の+10.40%は高市ラリーではない
- Physical AIは世界中で注目されるテーマ
- 材料出尽くしの影響を受けにくい
- **高市依存度が低い＝長期保有向き**

### 4. 「生成AIが決める」コンセプトの実践
**Day 11の教訓**:
- 「材料出尽くしかもしれない」は人間の主観
- 不確実性がある時こそ、AIの多数決に従う
- 感情を排除し、論理的判断に徹する
- 結果は市場が決める、AIは最善を尽くすのみ

- IHI 1日で+8.1%の急騰
- 防衛テーマの威力を実証
