# 生成AIに従って30日間株取引してみた

## 📖 プロジェクト概要

本プロジェクトは、生成AIのアドバイスに従って30日間株取引を行い、その過程と結果を記録・分析し、最終的に書籍化することを目指しています。

## 🎯 目的

- 生成AIを活用した株取引の可能性と限界を探る
- 日々の取引記録を詳細に残す
- AIとの対話を通じた投資判断プロセスを可視化する
- 実践的な知見を書籍としてまとめる

## 📁 プロジェクト構造

```
.
├── README.md           # このファイル
├── daily-logs/         # 日々の取引記録
├── notes/              # 気づきやメモ
├── manuscript/         # 原稿ドラフト
├── data/               # データや分析結果
└── prompts/            # 使用したプロンプト集
```

## 📅 実施期間

- 開始日: 2025年10月2日
- 予定期間: 30日間
- 記録方法: 毎日の取引内容、AI との対話内容、市場分析、感想などを記録

## 🤖 使用するAI

本プロジェクトでは、複数の生成AIを使用して銘柄選定を行い、その結果を比較検証します。

- **Claude Sonnet 4.5** (Anthropic)
- **Gemini Pro** (Google)
- **ChatGPT** (OpenAI)

## 💡 8日間で学んだこと

### 1. AIの特性理解と活用法
- **Claude**: 市場トレンド分析、マクロ視点、リスク管理重視
- **Gemini**: テクニカル分析、保守的判断、長期視点
- **ChatGPT**: バランス重視、戦略的思考、実践的提案

### 2. AI多数決の有効性と限界
**威力**:
- 感情を排除した客観的判断
- 複数の視点で多角的に判断
- Day 7-8で「休むも相場」を全AI一致で実践

**限界**:
- 最新情報がなければ誤った判断
- 人間による最終チェックが不可欠
- ファンダメンタルズ確認は必須

### 3. 損切りの重要性（Day 7の教訓）
**石川製作所の損切り**:
- -5.49%で機械的に損切り実行
- 感情に左右されず、ルールを守った
- プロジェクト初の本格的損切り
- 「損小利大」の原則を実践

**教訓**: 
- 損切りラインは厳格に守る
- 感情的な判断を排除
- これ以上の深傷を回避する勇気

### 4. 「休むも相場」の実践（Day 7-8）
**Day 7-8で売買なし**:
- 政局不安（自公連立崩壊）で不確実性最大
- 日経平均史上最高値圏でリスク高い
- 全AI一致で様子見判断
- 現金比率95%で次の機会を待つ

**教訓**:
- 30日チャレンジでも「待つ」ことの価値
- 無理に買わない判断力が重要
- 機動力確保（現金比率）の意義

### 5. 調整局面での対処法
**半導体株の乱高下**:
- Day 5で調整入り、Day 6で爆発的反発
- サムコ: +9.3% → +26.22% → +7.78%
- 野村マイクロ: +3.0% → +7.59% → -1.78%

**教訓**: 
- 調整局面で慌てて売却しない
- 本質的需要が変わらなければ保有継続
- ボラティリティはチャンス

### 6. 政治テーマ株の難しさ
**防衛株の教訓**:
- Day 3-4で高市ラリーに乗る
- Day 7で石川製作所を損切り
- Day 8で自公連立崩壊の衝撃
- IHIも-4.52%で苦戦

**教訓**:
- 政治テーマは諸刃の剣
- 政局リスクは予測困難
- グローバルテーマ（AI、半導体）の方が安定

### 7. Physical AIテーマの強さ
**安川電機の安定感**:
- Day 6で購入、初日±0%
- Day 7で+1.91%
- Day 8で+6.76%（政局不安の中で独り勝ち）

**教訓**:
- 国内政治に左右されにくいテーマ
- 長期的な技術トレンドの重要性
- ソフトバンクGの買収効果継続

### 8. 高値掴みを回避する勇気（Day 8）
**高市テーマ株の誘惑**:
- Day 6（10/6）に核融合・宇宙関連が急騰
- QPS研究所、助川電気工業など
- Day 8で高市首相就任は50-50の賭けと冷静に分析

**教訓**:
- 急騰後の高値掴みリスク
- 確率で考える投資判断
- 「待つ」ことも選択肢
